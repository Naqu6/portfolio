Designing a Modern Concept Management System

For years, the IDEO Toylab used a legacy, PHP system to store their toy concept information. Although there were effective tools, the database was becoming bloated and the system's age was showing. Currently, as a software engineer at IDEO, I'm responsible for designing and building the new system.

Avoiding Second System Syndrome

Because the old system had been in use for decades, the designers and managers were very accustomed to it. I wanted to ensure that the new system didn't lose any functionality and improved upon the issues that plauged the old system.

Initally, I did research into how the main users of the program used the software. I ensured that I fully understood how the software was used. This encompased spending time working with the old software to get an additonal perspective on what worked and what didn't.




Initial Development

My first task was to get the core functionality of the software written in a more modern language with a cleaner database structure. I decided upon using Python and Django. Django is a python library that utilizes a SQL database and allows software to be built quickly. Their motto is "the web framework for perfectionists with deadlines".

While building the initial product, I worked with interaction designers at IDEO. We designed a new interface and new workflows, keeping some of the old features as well as designing new ones.




Implementation

Once the leads were satisfied with the new product, the Toylab switched to using the new system. The old database was quite bloated, so writing the scripts to migrate over the media and database were large tasks. I spent a lot of time reading through legacy PHP code to understand how the database was formatted. Additionally, there was no consistent method in which the media was stored which added to the complexity of the migration script.

Currently, I'm working with the Toylab Invention Lead to designing new features. I'm leading development on more web features and an iOS mobile application.
